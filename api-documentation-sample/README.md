# API Documentation Sample

This sample demonstrates REST API documentation for a sample transaction service. It includes authentication, endpoint details, request and response examples, error codes, and troubleshooting guidance.

## Purpose

The purpose of this sample is to show how API information can be structured for developers and integration teams.

## Documentation Skills Demonstrated

- REST API documentation
- JSON request and response examples
- Authentication workflow
- Endpoint description
- Parameter description
- Error code documentation
- Developer-focused writing

## Sample Topics

- API overview
- Authentication
- Create transaction endpoint
- Check transaction status endpoint
- Error codes
- Troubleshooting
