# User Manual / User Guide Sample

This sample demonstrates end-user documentation for a software product feature. It includes a feature overview, prerequisites, task-based procedures, notes, warnings, expected results, and troubleshooting guidance.

## Purpose

The purpose of this sample is to show how complex product features can be explained in a clear, simple, and user-friendly way.

## Documentation Skills Demonstrated

- User-focused writing
- Task-based procedure writing
- Clear step-by-step instructions
- Notes, tips, and warnings
- Expected results
- Troubleshooting guidance
- Product feature explanation

## Sample Topics

- Overview
- Prerequisites
- Configuring the feature
- Verifying the configuration
- Troubleshooting common issues
